# Acid-Base Clinical Calculator

Interactive acid-base interpretation calculator for clinical teaching and exam review.

Features:

- pH / PaCO2 / HCO3 primary disorder workflow
- Winter formula and compensation range visualization
- Albumin-corrected anion gap
- Delta-delta only when confirmed HAGMA
- NAGMA pathway with UAG, urine pH, and RTA clues
- Osm gap / toxic alcohol support
- Lecture screenshot references and Davenport map point plotting

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
